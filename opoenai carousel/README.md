# OpenAI/Codex LinkedIn Carousel

This folder is separate from the Claude/Canva workflow.

## Files

- `sudan_humanitarian_ai_linkedin_carousel.pdf`  
  LinkedIn-ready carousel PDF upload.

- `slides/slide_01.png` to `slides/slide_09.png`  
  Individual 1080 x 1350 PNG slides.

- `generate_carousel.py`  
  Rebuilds the carousel from local project assets and chart images.

- `contact_sheet.png`  
  Quick preview of all 9 slides in one image.

- `linkedin_caption.md`  
  Suggested post caption.

## Suggested Posting Flow

1. Upload `sudan_humanitarian_ai_linkedin_carousel.pdf` to LinkedIn as a document post.
2. Use the caption from `linkedin_caption.md`.
3. Add the dashboard link after deployment.
4. Tag relevant humanitarian AI, data science, and social-impact communities.

## Notes

This carousel is careful not to claim operational deployment. It describes the project as a responsible AI prototype using synthetic demo data and public humanitarian context sources.
